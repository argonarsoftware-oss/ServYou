ServYou - VPS Deployment Guide

Salon • Spa • Clinic Booking System — Domain: servyou.argonar.co — VPS: 139.177.187.26

Step 1 — Namecheap DNS Setup

DNS Go to Namecheap → Domain List → argonar.co → Advanced DNS

Add this A record:

TypeHostValueTTL
A Recordservyou139.177.187.26Automatic
Note: DNS propagation can take 5-30 minutes. You can check with: ping servyou.argonar.co

Step 2 — Apache VirtualHost

SSH SSH into your VPS and create the config file:

Create the config file:

nano /etc/apache2/sites-available/servyou.argonar.co.conf

Paste this entire VirtualHost config:

<VirtualHost *:80> ServerName servyou.argonar.co DocumentRoot /var/www/ServYou/backend <Directory /var/www/ServYou/backend> AllowOverride All Require all granted </Directory> # Block .git access <DirectoryMatch "^/.*/\.git"> Require all denied </DirectoryMatch> # Block guide and sensitive files <FilesMatch "(guide\.html|\.env|deploy\.log)$"> Require all denied </FilesMatch> ErrorLog ${APACHE_LOG_DIR}/servyou-error.log CustomLog ${APACHE_LOG_DIR}/servyou-access.log combined </VirtualHost>

Save: Ctrl+O → Enter → Ctrl+X

Enable the site and reload Apache:

a2ensite servyou.argonar.co.conf && a2enmod rewrite && systemctl reload apache2

Step 3 — SSL with Let's Encrypt

SSL Install certbot (if not already installed) and get SSL:

Install certbot:

apt install certbot python3-certbot-apache -y

Get SSL certificate:

certbot --apache -d servyou.argonar.co
Note: When prompted, choose to redirect HTTP to HTTPS (option 2). Make sure DNS is already pointing before running this.

Step 4 — Fix Permissions for Webhook Deploy

PERMS Allow Apache (www-data) to run git commands:

chown -R www-data:www-data /var/www/ServYou/.git && git config --global --add safe.directory /var/www/ServYou

Step 5 — Set Up MySQL Database

DB Import the database schema:

mysql -u root -p < /var/www/ServYou/backend/sql/schema.sql

Verify database was created:

mysql -u root -p -e "USE servyou_db; SHOW TABLES;"

Step 6 — Pull Latest Code

GIT Pull the latest from GitHub (includes webhook-deploy.php):

cd /var/www/ServYou && git pull origin main

Step 7 — Test Everything

TEST Verify the API is working:

Test API endpoint:

curl -s -H "Authorization: Bearer changeme-servyou-2026" https://servyou.argonar.co/api/bookings/stats
Expected response: {"success":true,"data":{"total":"0","pending":"0",...}}

Test webhook (from your local machine):

curl -s https://servyou.argonar.co/webhook-deploy.php
Expected: Method Not Allowed (405 — means it's working, only accepts POST from GitHub)
